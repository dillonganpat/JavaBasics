Java Basics Projects

This repository contains a collection of Java programs demonstrating fundamental programming concepts, including object-oriented programming, conditionals, loops, input/output, and basic data types.

Description of Programs

VariablesDemo.java – Demonstrates primitive data types and Strings.

CircleCalculator.java – Calculates the area of a circle using user input.

ConditionalStatements.java – Determines if a number is positive, negative, zero, even, or odd.

Account.java – A simple class representing a bank account with deposit, withdraw, and toString methods.

Student.java & StudentApp.java – Demonstrates classes, constructors, getters, setters, and conditional logic.

SimpleChat.java – A basic interactive chat program using Scanner for user input.

Other files include practice exercises for loops, math operations, and input/output.

How to Run
git clone https://github.com/dillonganpat/JavaBasics.git

Open the project in IntelliJ IDEA (or your preferred Java IDE) and run the programs from the src/main/java/tt/LabOne/ folder.
